Sentinel — AI Scheduler

Аннотация проекта

Sentinel — интеллектуальный планировщик с мультимодальным вводом. Пользователь кидает скрин письма, фото списка или короткий текст, система извлекает задачи, дедлайны, приоритеты и формирует таймблоки с экспортом в Календарь. Свои модели не нужны, распознавание и интерпретация идут через внешние Vision/LLM API.

Ключевая особенность — модель Resource Battery: индикатор «энергии дня», учитывающий тип нагрузки, длительность и частоту переключений. На её основе работают Auto-Rest Nudges (мягкие вставки микро-перерывов), Achievements (достижения за устойчивые полезные паттерны) и Rebalance — кнопка, которая оптимизирует порядок дел внутри дня.

Цель проекта

Реализовать мобильное приложение "Sentinel"

Задачи проекта

1. Выполнить сравнительный анализ существующих аналогов.
2. Спроектировать и реализовать мобильное приложение для iOS с реализацией ключевого функционала.
3. Интегрировать внешний AI (Vision/OCR + LLM) для извлечения задач, дедлайнов и приоритетов из пользовательских данных.
4. Реализовать серверную часть (API) для обработки, нормализации данных и поддержки ключевого функционала.
5. Провести тестирование и доработку приложения на основе обратной связи.
6. Подготовить техническую документацию.

Задачи каждого из участников проекта (роли участников проекта)

1. Frontend/iOS-разработчик
• Проектирование и реализация пользовательского интерфейса на SwiftUI.
• Реализация мультимодального ввода (фото, текст, скриншоты).
• Интеграция с Apple Календарем.
• Реализация взаимодействия с API (отправка данных на обработку, получение ответов).
• Внедрение ключевого функционала: «Resource Battery», таймблоков, достижений и подсказок.
• Адаптация под разные устройства (iPhone, iPad, опционально — macOS, watchOS, visionOS).
• Тестирование и оптимизация фронтенда.

2. Backend/Python-разработчик
• Проектирование архитектуры и логики серверной части (FastAPI).
• Интеграция с внешними AI-сервисами (OCR, Vision, LLM).
• Разработка алгоритмов нормализации задач, расчёта ресурса, формирования расписания и системы достижений.
• Разработка и поддержка API для коммуникации с мобильным приложением.
• Работа с базой данных, хранением пользовательских данных и событий.
• Обеспечение безопасности и конфиденциальности информации.
• Развёртывание, документация и поддержка бэкенда.
